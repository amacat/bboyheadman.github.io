A string can be converted into an array using the .split() method. The string with be divided by whatever you pass it as a parameter in quote marks.

Let's try turning this greeting to you into an array!

var greet = "Hello !";
var greetSplit = greet.split(" ");
console.log(greetSplit);
["Hello", "!"]

Great work! You .split() the string by the empty spaces and turned the string into and array
