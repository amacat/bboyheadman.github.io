It's project time! This time, however, let's use an HTML file with a &lt;script&gt; tag in the body section.

Create a chapter variable, assign 0 to it and increase it by 2. Add a sentence to the project variable that includes chapter and display it on the document.

<!doctype html>
<html>
 <body>
  <script>
   var project = "I think this is the beginning of a beautiful project";

   var chapter = 0;
   chapter += 2;

   project += " and this is only chapter " + chapter + "!";
   console.log(project);
  </script>
 </body>
</html>

Whee, we can place scripts within the &lt;script&gt; tag! Let's keep this file and use it throughout the micro-course, okay?
