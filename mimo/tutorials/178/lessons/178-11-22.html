Objects can include functions. Here we have the car object constructor with a method inside which we can use to change the colour of the car

functioncar(make, color) {
    this.brand = make;
    this.color = color;
    this.changeColor = function (differentColor) {
      this.color = differentColor
    };
};
var myCar = ("Honda", "black");

We've set myCar.color to equal "black". Let's try changing the color

myCar.changeColor(""red""white""blue"")
console.log(myCar.color);
"red""white""blue"

Awesome! You changed the .color of myCar using it's internal method.
