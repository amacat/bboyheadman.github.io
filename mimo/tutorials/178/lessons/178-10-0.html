Using JavaScript with the HTML DOM we can change all the aspects of an HTML document.  The HTML DOM  is a standard for how we alter HTML elements.

We can alter parts of the HTML DOM by putting our JavaScript between &lt;script&gt;&lt;/script&gt; tags. 

<html>
<body>

<h1>Introducing the DOM</h1>

<p id="trial"></p>

<script>
document.getElementById("trial").innerHTML = "Writing JavaScript and getting a webview!";
</script>

</body>
</html>
