Unlike lists, we can initialize a dictionary while we declare it. That should save some time, right?

var dict = new Dictionary<string, int>() {
 {23, "Michael"},
 {91, "Dennis"}
};

Whoa! We can put the key-value pairs in braces after the constructor. {23, "Michael"} is such a pair: 23 is its key and "Michael" is its value.

Psst: again, if we initialize the dictionary right away, we can use var instead of Dictionary&lt;string, int&gt;.
