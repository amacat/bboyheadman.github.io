Accessing elements in a dictionary is a little different to arrays and lists. We access a value by using it's key.

Dictionary<string, int> dict = new Dictionary<string, int> ();
dict.Add("Ebenezer", 11);
dict.Add("Pippin", 45);
dict.Add("Estella", 37);
Console.Write(dict["EbenezerPippinEstella"]);
114537

See that? Not too hard to access the values!

A value won't always be unique to a key so there isn't a way to access a key by value. The best way to find a key is to use the .Keys method we'll see in a few lessons!
