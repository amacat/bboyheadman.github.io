Now, what if we didn't like having the commit and its revert in the log? Well, we can also reset to the v1 commit from earlier.

$ git reset --hard v1master 
$ git log
commit f8eff11
Author: Elliot <elliot@allsafe.com>
Date: Sat Oct 8 13:21:11 2016

Great! By giving v1 as an argument, the log only shows that commit and those that have come before it.
