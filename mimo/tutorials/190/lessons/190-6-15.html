Let's try looking at an alternative to merging: rebasing. First we need to return our branches to a point before any merges.

Looking back, commit 1b18076 was just before any merges.

$ git checkout master
$ git reset --hardrevert --hard 1b18076
HEAD is now at 1b18076 Wrote tests

Nice. Remember to use reset --hard with the commit hash to reset back to a particular commit.
