To check differences between a file on our repository and in our working directory we can use a special git command.

$ touch 1.txt
$ touch 2.txt
$ git add .
$ git commit -m "Add 1.txt and 2.txt"
$ echo We are fsociety > 1.txt
$ echo Hello World > 2.txt

We've just created two text files and added some text after committing the files. Let's try to see the changes since our last commit.

We can either use git diff and a filename to see it's differences on the repo and working directory or git diff with the branch name to see the differences in all files.
