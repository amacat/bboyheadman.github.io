We can move files around with terminal commands but we can do it even quicker using Git commands!

Let's move User.java from our current directory to a new one!

$ mkdir src
$ git movemv User.java src
$ git status

On branch master
Untracked files:  (use "git add <file>..." to include in what will be committed)
 src/

By using the git mv command, we can add src/User.java to the repo and remove the original User.java from the repo in one command!
