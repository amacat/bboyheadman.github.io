Let's continue the that project from earlier on, shall we?

Create a CHANGEME.md file, stage it and then remove it from the staging area. Then create a COMMITME.md file, stage and commit then undo the whole commit.

$ touch CHANGEME.md$ git add CHANGEME.md$ git checkout CHANGEME.md$ touch COMMITME.md$ git add COMMITME.md$ git commit -m "Create COMMITME.md file"$ git revert HEAD

Awesome! In your journey as a developer, being able to undo and reset changes will prove invaluable when working on more complex projects.
