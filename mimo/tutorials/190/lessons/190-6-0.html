Now, let's work on a little project to reinforce what we've talked about so far!

Let's create a project directory and turn it into a repository. In there, create a fortune_cookie.txt file, stage it and commit it with a message.

$ mkdir project$ cd project$ git init$ touch fortune_cookie.txt$ echo LEAVE ME HERE > fortune_cookie.txt$ git add fortune_cookie.txt$ git commit -m "Create a fortune cookie"

Great job! If we keep up that pace, we'll be a Git gurus in no time.

Think of the Git workflow: we make changes before staging and stage changes before committing.
