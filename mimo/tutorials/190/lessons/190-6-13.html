We can also look at a particular commit to see the normal log details as well as the diff in that commit.

$ git showshow f871379 
commit f871379
Author: Elliot <elliot@allsafe.com>
Date: Tue Nov 15 15:17:59 2016 +0100
   
    Add 1.txt and 2.txt

diff --git a/1.txt b/1.txt
new file mode 100644

diff --git a/2.txt b/2.txt
new file mode 100644
commit f871379
Author: Elliot <elliot@allsafe.com>
Date: Tue Nov 15 15:17:59 2016 +0100

Phew, pretty long output there. But as we can see, using git show will show us the most recent commit and the diff for the files in that commit.

Using git show with a commit hash, we can see the details of that commit along with the diff for the files in that specific commit.
