$ git history --all  * 1b18076 2015-06-24 | Wrote tests (HEAD -> new_branch) [Elliot]

* f8eff11 2015-06-24 | Created README.md [Elliot]* 1b18076 2015-06-24 | Wrote tests (HEAD -> new_branch) [Elliot]

See that? Our commit history is a lot easier to read because all the commits are on the same branch.

 It doesn't make a difference if we use --all or not.
