Let's give our branch merging skills a whirl!

Create a new branch, add something to cookie.txt and make a commit there. Back in master, merge the new branch into master.

$ git branch new_branch
$ git checkout new_branch
$ echo secret > cookie.txt
$ git add .
$ git commit -m "Modify cookie.txt"
$ git checkout master
$ git merge new_branch

Super! Merges and conflicts are really tricky at first but they're important to understand. Being able to deal with them is a key skill of a developer!
