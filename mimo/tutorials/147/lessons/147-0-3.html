In order to create these HTML files, we need an HTML editor. If you’re on a Mac, TextEdit will do the job; if you’re on a Windows computer, Notepad is fine.

Try to create a plain-text file and type in something like this:

<html>
 <body>
  I can be anything you want.
 </body>
</html><body>I can be anything you want.</body>
</html>
]]>

Then save the file with a .html extension instead of .txt and open it in your web browser.
