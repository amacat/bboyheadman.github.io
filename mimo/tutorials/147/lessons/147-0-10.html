Let's talk about a tag at the beginning of a web page that we've neglected so far: the doctype, which tells the browser about the version of HTML we're using.

Can you help me set the doctype to HTML5, the current version of HTML?

<!doctype htmlxml>
<html>
 <head>
  <title>My web page</title>
 </head>
 <body>
  Hello!
 </body>
</html>
<html>
<head>
<title>My web page</title>
</head>
<body>Hello!</body>
</html>
]]>

Nice! &lt;!doctype html&gt; means that we're using HTML5. If we skip the doctype, the browser has to guess, which is never a great idea.
