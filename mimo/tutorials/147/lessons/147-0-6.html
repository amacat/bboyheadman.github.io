Let's start with the &lt;html&gt; tag, which denotes the beginning and the end of a web page.

Do you know what that means?

<html><!-- Web page content --></html>

Sweet! The &lt;html&gt; tag, which starts with &lt;html&gt; and ends with &lt;/html&gt;, is the container for everything that belongs to a web page.

Think of a sandwich: everything that belongs to a web page needs to go between &lt;html&gt; and &lt;/html&gt;.
