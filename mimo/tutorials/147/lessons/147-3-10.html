Now that we know how to include multimedia in a web page, it's time for your blog to grow up.

Inside the introduction paragraph, add an image of you with width and height attributes. Below, create an inline frame with your favorite YouTube video.

<!doctype html>
<html>
 <!-- Things from earlier on -->
 <body>
  <h1>Welcome</h1>
  <p>
   <imgsrc="https://getmimo.com/r/profile.png">
   I'm <strong>Henry</strong> and this is a blog about <em>my life</em>.
  </p>
  <iframe width="420" height="315" src="https://www.youtube.com/embed/Vhh_GeBPOhs"></iframe>
  <!-- Things from earlier on -->
 </body>
</html>
<html>
<head>
<title>My blog</title>
</head>
<body><h1>Welcome</h1><p><img src="https://getmimo.com/r/profile.svg">I'm <strong>Henry</strong> and this is a blog about <em>my life</em>.</p><iframe width="100%" src="https://www.youtube.com/embed/Vhh_GeBPOhs"></iframe><hr><a href="https://fb.com/hgllnt" target="_blank">Add me</a><a href="https://twitter.com/hgllnt" target="_blank">Follow me</a></body>
</html>
]]>

It's still not the prettiest blog on Earth but it just got a lot richer in content.
